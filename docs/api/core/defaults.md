# core.defaults

Centralised default constants for taskclf.

Path constants (`DEFAULT_DATA_DIR`, `DEFAULT_MODELS_DIR`, etc.) are
absolute paths derived from `taskclf_home()`.  Set the `TASKCLF_HOME`
environment variable to override the base directory; see
[core.paths](paths.md) for the full resolution order.
